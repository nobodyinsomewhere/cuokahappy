# Tavern Card Generator 

> 一个纯前端、开箱即用、可直接部署的酒馆角色卡生成工具。  
> 支持 JSON / PNG 角色卡导入导出、AI 一键补全设定、多主题切换与本地自动保存。

---

## 简介

Tavern Card Generator 是一个面向角色卡创作场景的轻量级前端工具。  
它不依赖后端、不需要数据库，也不要求复杂的环境配置。

你可以直接本地打开使用，也可以部署到 Netlify、Vercel、Cloudflare Pages 等静态托管平台，作为一个可长期维护的在线工具站。

## 使用方式

### 本地运行

将以下文件放在同一目录下后，直接打开 `index.html` 即可：

- `index.html`
- `style.css`
- `app.js`
- `png-card.js`
- `storage.js`
- `utils.js`
- `api.js`

### 在线部署

本项目可以直接部署到静态托管平台，例如：

- Netlify
- Vercel
- Cloudflare Pages

以 Netlify 为例：

1. 将项目上传到 GitHub 仓库
2. 在 Netlify 中选择 **Import from Git**
3. 连接对应仓库
4. 完成部署后即可在线访问

## 数据存储说明

本项目默认**不会将用户数据保存到你的服务器**。

用户创建的角色数据主要存储在：

- 浏览器本地存储（`localStorage`）
- 用户自己导出的 JSON / PNG 文件

这种方式的优点是：

- 更轻量
- 更易部署
- 更偏本地优先
- 不需要后端维护
- 对用户隐私更友好

同时也建议用户定期导出角色卡文件作为备份，以避免清理浏览器缓存后导致本地数据丢失。

然后使用 **AI 补全设定** 功能自动生成或补全文本内容，例如：

- summary
- description
- personality
- scenario
- first message
- example dialogue
- creator notes

## 注意事项

- 默认不包含后端、数据库和账号系统
- 如果你需要云端存储、账号登录、多人同步等功能，需要额外接入后端服务
